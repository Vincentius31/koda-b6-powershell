1. Copy-Item (cp) = mengcopy sebuah file directory
2. Get-Content (cat) = menampilkan konten yang ada pada sebuah file
3. Rename-Item (rni) = mengganti nama folder atau file
4. Set-Location (cd) = mengganti directory file
5. New-Item (ni) = membuat file baru
6. Move-Item (mv) = memindahkan file atau folder
7. Clear-Host (cls) = menghapus semua yang ada pada layar
8. Test-Path (path) = mengecek apakah file tersebut masih ada atau tidak
9. Get-Service (ps) = menglist semua proses yang sedang berjalan
10. Test-Connection (ping) = mengecek koneksi internet
11. Restart-Service = mengrestart segala proses yang sedang dijalankan
12. Get-Command (gcm) = list semua command dan function 
13. winget = mendownload aplikasi yang sudah terintregasi dengan windows
14. Get-History = mengembalikan command yang sudah dituliskan 
15. ConvertTo-HTML = membuat file HTML
16. Rename = merubah nama file yang akan diubah
17. Sort-Object (sort) = mengurutkan file dengan parameter nilai sebuah file
18. Set-Variable (set, sv) = menetapkan sebuah nilai dari sebuah variable
19. Group-Object = mengkelompokan sebuah file dengan tipe yang sama
20. Remove-Item = menghapus file yang dituju
